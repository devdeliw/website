I study applied math and astrophysics at UC Berkeley. I learn to write fast and elegant numerical code. Currently I am developing COre Rust Architecture for Linear Algebra (CORAL). It’s a pure-Rust implementation of Basic Linear Algebra Subprograms (BLAS) for AArch64. It’s speed is well-comparable to industry BLAS like OpenBLAS. You can see the GEMM benchmark here.

The purpose of this website is to store a record or “blog” of how I write and reason about code. I plan to add documentation here as well.
