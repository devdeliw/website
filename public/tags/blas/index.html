    What is CORAL?

    October 9, 2025
    Software
    BLAS

    In the past few months I’ve become immensely interested in scientific computing and writing fast code. I started CORAL as a project to learn both at the same time.

    CORAL stands for COre Rust Architecture for Linear algebra. It is an implementation of the Basic Linear Algebra Subprograms, or BLAS in pure Rust, but only for AArch64 architectures right now.
